# HealthScaleAnalyzer
HealthScale Analyzer — это десктопное приложение, разработанное для автоматизации процесса оценки физического состояния людей, выявляя нуждающихся во внимании со стороны специалистов.

Главная цель — упростить рутинную работу по обработке антропометрических данных и помочь своевременно выявлять людей с отклонениями в физическом развитии для последующей корректирующей работы.

## Функционал

- Расчет Индекса Массы Тела (ИМТ) с классификацией по возрастным нормам.
- Формирование и экспорт Excel-отчета, включающего следующее:  
    - общая статистикой по всем людям, которые были в исходных данных (количество людей с определенной группой здоровья, физкультурной группой и определенными оценками ИМТ)
    - лист с информацие о каждом человеке
- Система предоставляет графический интерфейс (GUI) для выбора исходных Excel-файлов, в которых заполнены ФИО, дата рождения, вес, рост(см), опицонально группа здоровья и группафизкультуры.
